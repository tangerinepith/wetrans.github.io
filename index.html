About Wetrans

Wetrans is a desktop computer-aided translation software. Based on translation memory and term database technology, it provides a set of integrated tools for rapid creation, editing and proofreading of high-quality translation.

Its basic principle is translation memory + machine translation + artificial translation. At the same time, the computer-aided translation is based on the memory, and the system works on the basis of the memory.

Services

The overall functions include user management, term library management, translation memory management, vocabulary weight management and project information management.

Desktop applications

Wetrans is a Windows computer software, currently working in the computer desktop, the follow-up will continue to develop web version and other more modules.

User info

User information management mainly includes four basic functions: user login, user registration, user name modification and user password modification.

Term base

A term bank is a collection of terms used to express concepts in a specific discipline, which can be either words or phrases,and managed by specialized term workers.

Translation memory

Translation memory is the core technology of computer-aided translation software. Memory is a local file accumulated for a long time.

Weight vocabulary

By importing a certain word and its weight into Wetrans, the system can carry out weighted calculation, so as to improve the accuracy of system matching translation sentence pairs.

Project info

The core business function module of Wetrans is the project information management function module, and the other four modules provide services for this module.
